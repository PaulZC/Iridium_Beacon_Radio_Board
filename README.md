# Iridium 9603N Beacon Radio Board

A radio board for the [Iridium 9603N Beacon](https://github.com/PaulZC/Iridium_9603_Beacon). Equipped with an LPRS eRIC4 or eRIC8 integrated radio module,
this board allows remote control of radio-enabled devices near the Iridium Beacon (e.g. a radio-enabled cut-down device).
